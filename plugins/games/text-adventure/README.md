# @zhin.js/plugin-text-adventure

交互式**文字冒险** — 31 个可探索区域、15 种道具、15 种结局，完整流程 50+ 步。

## 规模

| 维度 | 数量 |
|------|------|
| 可探索区域 | 31 |
| 终局结局 | 15 |
| 可收集道具 | 15 |
| 预估最长流程 | 50+ 步（含往返探索） |

## 命令

`冒险` · `冒险 开始` · `冒险 继续` · `冒险 地图` · `冒险 成就` · `冒险 放弃`

（英文别名 `adv` 仍可用；也可发送 **`游戏`** 从大厅进入）

## 进度与成就（用户绑定）

探索进度与成就按 **玩家 ID** 持久化（`adv_profiles` 表），跨频道、跨局累计：

- 场景内顶部显示：`🗺️ 区域进度 · 🎬 结局图鉴 · 🏅 成就`
- `adv map` — 分区地图解锁（✅/⬜）+ 结局图鉴
- `adv achievements` — 15 项生涯成就
- 达成新成就时，终局场景会弹出解锁提示


## 完美结局路线（剧透）

1. 碎石：火把 + 短剑 → 石塔测绘 + 星图 + 星象台罗盘
2. 走廊：壁画 → 低语画廊聆听 → 精灵泉对话 + 饮水
3. 秘径庭院：草药 → 温室灵药 → 书库卷轴/钥匙 → 锻炉淬剑
4. 钟楼取铃 → 墓穴驱影 → 骸骨甬道卷轴破机关 → 龛室答「星辰坠落」
5. 积水厅 → 市集 → 冥桥 → 晶洞月华/银羽 → 月池星图共鸣 → 沉殿（可选星守）
6. 岔口回音巨门或封印密室 → 光影门 → 王座厅星辉归位

改 `src/story*.ts` 后执行 `pnpm --filter @zhin.js/plugin-text-adventure build`。
