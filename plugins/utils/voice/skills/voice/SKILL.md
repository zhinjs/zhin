# 语音转换技能

## 工具

| 工具 | 用途 | 说明 |
|------|------|------|
| `voice_stt` | 语音转文字 | 将语音消息识别为文字 |
| `voice_tts` | 文字转语音 | 将文字合成为语音消息发送 |

**Example:**
```
用户: [发了一条语音] 这说的什么？
→ voice_stt(语音消息)

用户: 帮我把这段话念出来：今天天气不错
→ voice_tts(text="今天天气不错")
```

## 注意

- STT 和 TTS 能力受插件配置中的开关约束，可能被管理员禁用。
- 语音识别准确度取决于后端 STT 服务的质量和语音清晰度。
