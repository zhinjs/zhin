# 群日常分析

基于收件箱数据自动分析群活跃度。无 AI 工具可调用。

## 能力

- 统计群消息数量、参与人数、活跃时段分布
- 支持按群启用/禁用分析功能
- 定时任务自动生成分析报告推送到群内
- 通过聊天命令管理分析设置
