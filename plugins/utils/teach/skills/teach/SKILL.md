# 自定义问答查询技能

## 工具

| 工具 | 用途 | 说明 |
|------|------|------|
| `teach_query` | 查询问答库 | 搜索已有的问答对，支持关键词过滤 |
| `teach_stats` | 问答统计 | 当前问答库的数量、命中频率等 |

**Example:**
```
用户: Bot 学了哪些问答？
→ teach_query()

用户: "你好"这个问答被触发了多少次？
→ teach_stats(keyword="你好")
```

## 注意

- 问答的教学（`教学 问题 回答`）和删除（`忘记 问题`）通过聊天命令操作，不是 AI 工具。
- 支持精确匹配和正则匹配两种模式。
- 问答触发有冷却时间配置，防止刷屏。
