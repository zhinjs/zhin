# 沙箱测试适配器

本地开发调试专用。通过浏览器 Web UI 模拟聊天，无需第三方平台凭据。无 AI 工具可调用。

## 使用场景

- 开发新插件时在本地模拟消息收发
- 调试命令路由和中间件逻辑
- 依赖 HTTP 服务和 Console 插件提供 Web 界面
