## 快速上手
### 1. 初始化包管理器
```shell
npm init -y # 初始化 package.json 文件
npm install typescript -D # 安装ts开发环境依赖
npx tsc --init # 初始化 tsconfig.json 文件
```
### 2. 安装zhin
```shell
npm install zhin
```
### 3. 初始化
```shell
npx zhin init --key  # 
```
